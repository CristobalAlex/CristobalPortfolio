## Alexandra Nicole Cristobal | Portfolio
A modern, responsive portfolio website built with **Vue.js 3** and **Tailwind CSS**. It features a dynamic **Dark/Light Mode** theme with a unique holographic lavender aesthetic, smooth animations, and showcases my projects as a Full Stack Web Developer.
**Live Demo:** [https://cristobalalex.github.io/CristobalPortfolio/](https://cristobalalex.github.io/CristobalPortfolio/)
## Features
* **Powered by Vite:** Ultra-fast development server and optimized build.
* **Dynamic Theming:**
    * **Dark Mode:** Deep Galactic Violet background with neon purple accents.
    * **Light Mode:** Soft Pastel Lavender background with holographic gradients.
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.
* **Tech Stack:**
    * [Vue.js 3](https://vuejs.org/) (Composition API)
    * [Tailwind CSS](https://tailwindcss.com/)
    * [FontAwesome](https://fontawesome.com/) (Icons)
